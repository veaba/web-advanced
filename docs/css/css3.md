# CSS3

## css3

### anination-timing-function

- `linear` 匀速
- `ease` 默认，低速开始，加快，结束前变慢
- `ease-in` 低速开始
- `ease-out` 低速结束
- `ease-in-out` 动画以低速开始和结束
- `cubic-bezier(n,n,n,n)` 可能值范围 0 - 1

### padding-top

- 默认值，0

### maring-top

- 默认值 0

### border-top-width

- 默认值 medium 定义中等上边框

### outline-width

- 默认值 medium 定义中等上边框
