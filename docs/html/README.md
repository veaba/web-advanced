- 带 id 属性的 DOM 元素有什么副作用？

- `clientWidth`：`height` + `padding`
- `offsetHeight`: `height` + `paddiing` + `border`
- w3c 标准盒模型中，默认情况下，块元素的总宽度 = content
- HTML5 事件可以触发多次

### 标准盒子模型
标准盒子模型宽度: content

- IE盒子模型: (margin+padding+border)*2+width
- 默认使用 content-box
