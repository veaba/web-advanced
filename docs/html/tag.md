## empty element (空元素)

### 总结
- [Empty element](https://developer.mozilla.org/zh-CN/docs/Glossary/%E7%A9%BA%E5%85%83%E7%B4%A0)
- 通常在一个空元素上使用闭合标签是无效的
- 不存在子节点（文本或内嵌元素）的元素
