# 文档

V8是Google的开源高性能JavaScript和WebAssembly引擎，用C++编写。它用于Chrome和Node.js等。

此文档旨在针对那些希望在应用程序中使用V8的C++开发人员，以及对V8的设计和性能感兴趣的人。 本文档向您介绍了V8，而其余的文档则向您展示了如何在代码中使用V8，并描述了它的一些设计细节，还提供了一组用于测量V8性能的JavaScript基准。
